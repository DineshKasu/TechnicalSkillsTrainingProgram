User guide for Installation

First Step

1. Unzip the file "TSTP" i.e Project source code (If it is zipped still)

Second Step

1. Install XAMPP application from the below link 
      https://www.apachefriends.org/download.html

2. Run the XAMPP AND click the "start" button of Apache and MySQL

Third Step

1. Copy the "TSTP" folder to the default directory which is
	c:/xampp/htdocs

Fourth Step (installing of Database)

1. Open Browser, preferred browser is GOOGLE CHROME (Optional is Firefox)

2. In the address or URL bar, copy and paste this line: http://localhost/phpmyadmin/ 

3. In the LEFT SIDE, find and click "New" or click the "Databases" menu
	then type the database name which is "tstp"

4. Locate the "tstp" in the left pane then click

5. click the "Import" menu and locate the directory of the database where you unzip it
                
           i.e FINAL_TSTP.SQL
 
6. click "GO" button to finish installing
    note: don't close the tab

Fifth Step (Opening the system)

1. Open new tab,

2. In the URL bar just type or copy and paste  

     http://localhost/tstp 
3. Done
